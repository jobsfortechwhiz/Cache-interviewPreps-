# Blogger Archive

Permanent backup of Blogger posts and pages to GitHub using Google Apps Script.

---

## Features

- Sync Blogger Posts
- Sync Blogger Pages
- Permanent GitHub Backup
- Automatic Metadata Generation
- Weekly Scheduled Sync
- Restore Ready Architecture
